Rectball

Rectball is a puzzle game for Windows, Linux, MacOS X and Android where the objective is to make the most amount of points before the time is over by tapping colour circles and making rectangles with them.

Download latest version Play Store

Current version: v0.0.10 • View on GitHub • Older releases • Report bugs

Play Store version will be released in the near future. In the meantime you can manually download and install the APK release into your phone. It is up to you whether you want to enable third-party sources on your phone.

The game is in Alpha stage at the moment. Please, report bugs at the tracker.
